# Visual references

Android screenshots are reference fixtures for the MonoGame iOS/Android UI clone. They are not packaged runtime assets.

| File                                     | Scene                        | Purpose                                                                  |
| ---------------------------------------- | ---------------------------- | ------------------------------------------------------------------------ |
| `android-main-menu-first.jpg`            | Main menu, first menu        | Logo/menu/music-control placement, profile/version/download affordances. |
| `android-song-select-hard.jpg`           | Song select                  | Base song list, selected difficulty panel, bottom controls.              |
| `android-song-select-personal-best.jpg`  | Song select + personal best  | Scoreboard overlay/state.                                                |
| `android-song-select-local-scores.jpg`   | Song select + local scores   | Local score list state.                                                  |
| `android-song-select-filter-overlay.jpg` | Song select + filter overlay | Search/sort/filter overlay geometry over song list.                      |
| `android-options-general.jpg`            | Options, General             | Source reference for OptionsScene full-screen settings parity.           |
| `android-mod-select.jpg`                 | Mod select                   | Full-screen dark overlay, mod grid, bottom stat strip.                   |
| `android-mod-custom-speed.jpg`           | Mod customize                | Custom speed panel and underlying mod grid dimming.                      |
| `android-collections-dialog.jpg`         | Collections dialog           | Large rounded dialog and list/search proportions.                        |
| `android-profile-alert.jpg`              | Profile alert                | Alert dialog styling and button rows.                                    |
| `android-exit-confirm.jpg`               | Exit confirm                 | Main menu modal dimming and confirm button layout.                       |
| `android-about-dialog.jpg`               | About dialog                 | Version/about modal typography, links, and button row.                   |

Current Android reference resolution: `2340x1080`.
